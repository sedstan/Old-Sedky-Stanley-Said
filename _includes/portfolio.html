Camp Social

This mobile first web app is a social and review camping site. It uses a web crawler to browse two other sites and store in the back-end and is delivered to the front-end. It is built with the MEAN stack. The front-end uses a boot-strap template.


    • Unicorn

    This web app is a site to search for workspaces or startups. This three person group project uses the Google Maps/Places APIs to fetch data to add to our API. It is then delivered to our front-end. This project was built with MongoDB, Express and Node on the back-end. We used Materialize on the front-end coupled Grunt to compile our Sass into CSS 3 and HTML 5.


    • Just Had

    This mobile first web app is a social networking app for Foodies. It allows a user to upload a photo and comment on a meal or drink they just had. The user can see all of the suggestions made by other users and the location of the meal. Additionally, it provides the user with suggestions in five mile radius of the post. The app was built using Ruby on Rails with Foundation on the front-end.


    • Word Scuttle

    This web app is a simple JavaScript, single-player word challenge game. The user is dealt a hand and must make a word using no more than seven letters.
